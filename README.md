# CRA + SSR Boilerplate

-   create-react-app (ejected)
-   express
-   code splitting(@loadable/component)
-   redux-saga
-   react-router-dom
-   react-helmet

## Install Packages

```shell
npm install
```

## Build

```shell
npm run build
npm run build:server
```

## Start

```shell
npm run start:server
```
